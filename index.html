Machine Learning based Li-Ion Cell Model

Reviewed literature on state of health estimation and charge balancing inaccuracies due to coulomb counting. Performed data extraction, visualization, implementation of different ML methods (K-Nearest Neighbor, Support Vector, Decision Tree, Random Forest Regressor) for comparison and forecasting of battery aging of multiple cells on Lithium Ion battery Aging & Testing data published by Sandia National Laboratories. Reported observable trends on battery degradation data under varying external parameters (Charge/Discharge Current, Temperature etc.)

Hand Gesture Controlled Drone

My work in the project was classifying hand gestures into 7 classes where I started with the basic step of extracting Cartesian coordinates of 21 hand landmarks for each 7 different gestures using MEDIAPIPE pre-trained model, and trained ANN model on the coordinates from custom dataset and considered the coordinates as features. Then, predicted gestures in real time using the weights of the ANN classifier, which was then used to control a drone simulated on Pybullet.
